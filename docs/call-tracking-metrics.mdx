# Ours Privacy Integration with CallTrackingMetrics

This guide will walk you through integrating **Ours Privacy** with your [CallTrackingMetrics](http://calltrackingmetrics.com/) account. You'll learn how to track calls, SMS, and form submissions seamlessly.

## CallTrackingMetrics Webhooks (custom integration)

For advanced tracking and to access detailed event data, you can set up a custom integration with CallTrackingMetrics. Follow these steps:

1. **Create a New Source in Ours Privacy**:

   * In the Ours Privacy app, navigate to **Sources** and create a new source.
   * Select **CallTrackingMetrics** from the options and copy the Webhook URL provided.

2. **Copy/Paste Your Ours Privacy Webhook Into CallTrackingMetrics**

   * Select the trigger that works for the event you are trying to push into Ours Privacy
   * Use the default Log Data event payload type.
   * **Note**: You can customize the payload structure in CallTrackingMetrics if needed. Ours Privacy allows you to map custom fields through the field mapping interface.

![](/images/docs/fa697f982dcc617c46b040b0482cdef56a286e076c93c4b13564e1ca9f26591e-Xnapper-2025-09-26-11.52.56.png)

3. **Submit a Test Event**:

   * Trigger a test event by clicking the "Test" button on your webhook in CallTrackingMetrics
   * Note: You have to first make sure that you have a call, message or event in CallTrackingMetrics. They do not let you send test data without at least one real call first.

4. **Map Extra Event Fields**:

   * Use the field mapping interface in Ours Privacy to map CallTrackingMetrics event data to Ours Privacy attributes.

### Payload Preview

If you are customizing your CallTrackingMetrics integration, you can use the "Preview Data" panel to view the latest payloads sent from CallTrackingMetrics to Ours Privacy.

### Key Benefits of the Ours Privacy CallTrackingMetrics Integration

* **Privacy Support**: Ideal for healthcare and other privacy-sensitive industries.
* **Custom Event Flexibility**: Track calls, SMS, form submissions, and more.
* **Enhanced Data Accuracy**: Server-side integration ensures reliable analytics.
