# LiveIntent Integration

LiveIntent is a marketing platform that helps advertisers connect their marketing data across multiple channels. With this integration, you can send events to LiveIntent via their Conversions API to track website events, app events, and offline conversions, enabling unified marketing data management and campaign optimization.

## Overview

This feature lets you configure LiveIntent as a destination to send marketing events and conversion data via their Conversions API, simplifying your technology stack by consolidating multiple data sources into a single connection.

## Use Cases

* Send website events and user interactions to LiveIntent
* Track app events and mobile user behavior
* Transmit offline conversion data from CRM systems
* Consolidate marketing data from multiple sources
* Optimize advertising campaigns with unified event data
* Track business messaging events and customer touchpoints

## Requirements

* A LiveIntent account
* Authorization token from your Account Manager
* Customer ID from LiveIntent
* Pixel ID for Conversions API

## Setup

1. Contact your LiveIntent Account Manager to obtain:

   * Authorization token
   * Customer ID
   * Pixel ID for Conversions API

2. Enter these credentials in your Ours Privacy integration settings

3. Configure event mapping for the data you want to send

## Additional Information

For more information about LiveIntent and their Conversions API, contact your Account Manager or visit the [LiveIntent website](https://www.liveintent.com/).
